# Personal Website

This is my personal website project, created to showcase my skills and provide information about me. The website is fully responsive and designed with simplicity and usability in mind.

## Pages Included

- **Home Page**: Provides an overview of the website, welcoming visitors and introducing the main content.  
- **About Page**: Shares information about me, including background, education, and skills.  
- **Contact Page**: Allows visitors to get in touch through a contact form or by providing social media and email information.

## Features

- Clean and modern design  
- Responsive layout for mobile and desktop devices  
- Navigation menu for easy browsing between pages  
- Interactive elements using JavaScript (e.g., form validation or dynamic content)

## Technologies Used

- **HTML**: Structure of the website  
- **CSS**: Styling and layout  
- **JavaScript**: Interactive functionality and dynamic content
